# AutoRename

为专有用途定制的Windows桌面应用

## 软件功能

1. 扫描用户的 Video 目录下的 mkv 文件（不包含子目录）
2. 筛选出其中文件名符合 `YYYY-MM-DD hh-mm-ss.mkv`的文件（例：`2025-09-09 11-32-39.mkv`）
3. 根据用户选择，进行重命名。命名规则为 `YYYYMMDD_<<LessonNo>_<LessonTag>.mkv`
4. 移动该文件到 `D:\Record\YYYYMMDD\` 目录下（例：`D:\20250909`）

## 交互逻辑

- 软件主界面为左右分栏
- 左侧为控制栏，包含一个日期输入框，一个序号输入框，一个标签输入框；一个确认按钮，一个跳过按钮
  - 从文件名中识别出日期，然后作为日期输入框的默认值
  - 序号为整数类型，范围为 1~9
  - 标签输入框候选值为：
    - 语文
    - 数学
    - 英语
    - 物理
    - 化学
    - 生物
    - 体育
    - 思政
  - 按下确认按钮以后，根据用户配置进行重命名和移动
  - 按下跳过按钮以后，则将这个文件添加到跳过列表中
- 右侧为预览栏。展示一个视频播放界面（h5默认的即可）
- 打开软件即显示主界面，主界面中显示任意一个未跳过，且符合要求的文件

## 实现细节

- 需要在 Video 目录下维护一个 `ignore.json`，用于储存选择跳过的文件名列表。
- 请使用 electron 进行实现
